# ZilMate Web Command Center

This folder contains a comprehensive web-based design for the ZilMate CLI.
It aims to provide an "Airy Minimalist" interactive interface that mirrors the features of the terminal-based CLI but with a more beautiful and modern aesthetic.

## Directory Structure

- `html/index.html`: The main interactive prototype.

## Design Principles

- **Aesthetic**: Airy Minimalist.
- **Color Palette**:
  - Surface: White (#FFFFFF)
  - Sidebar/Secondary: Faint Peach (#FFF5F0)
  - Primary Action: Rose Gold (#8E4A55)
  - Accent: Muted Rose (#C97B86)
- **Typography**: Inter (Clean, geometric, professional).
- **Shapes**: High corner radius (16px) for cards and containers, Pill-shaped buttons.

## Features Included

- **Interactive Chat**: Simulates the `zilmate chat` experience.
- **Swarm Dashboard**: Visualizes the 3-tier hierarchical swarm (CEO -> COO -> Departmental Heads).
- **Background Jobs**: Monitor queued tasks.
- **Digital Memory & Wiki**: Access long-term agent memory and corporate knowledge.
- **System Doctor**: Live health checks for dependencies (Playwright, rembg, etc.).
- **App Connectivity**: Status of Composio toolkits (GitHub, Slack, etc.).
- **Configuration**: Setup wizard interface.

## How to View

Simply open `html/index.html` in any modern web browser.
